Este repositório contém exercícios do curso "Lógica de programação: praticando com desafios" da Alura.

---

Cada exercício no site te acompanha para resolver cada um de acordo com as instruções das videoaulas. Eu primeiramente decidi fazer cada um de forma independente e depois ir ver a resolução de cada um, ajustando conforme o necessario e comentando meu raciocinio.

Abaixo listo qual faze do meu planejamento de resoluçao para esses exercicios e minha progreção. 

1. Resolver o exercício sem consultar soluções externas [CONCLUIDO]
2. Adicionar comentários em todos os exercícios [EM ANDAMENTO]
3. Revisar soluções comparando com respostas oficiais [A FAZEER]
4. Comparar abordagens e refatorar códigos [A FAZEER]
5. Padronizar estilo de comentários e refinar os comentários para referência futura [A FAZEER]

---

## ATENÇÃO ##

Este repositório é como um caderno durante meu aprendizado — mais importante que a resposta correta pra mim é meu raciocinio durante cada exercicio.
